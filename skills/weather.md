# Habilidad: Consulta de Clima

Permite al asistente obtener información meteorológica en tiempo real para cualquier ubicación.

## Capacidades:
- Consultar la temperatura actual, humedad y estado del cielo.
- Obtener pronósticos de lluvia o nieve.

## Cuándo usar:
- Cuando el usuario pregunta por el clima en una ciudad específica.
- Para tareas programadas (Cron) que requieren enviar el reporte del clima automáticamente.

## Herramientas relacionadas:
- `get_weather`: Obtiene los datos meteorológicos actuales.
