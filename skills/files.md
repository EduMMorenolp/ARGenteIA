## Gestión de Archivos

- Usá `read_file` para leer archivos de texto plano. NO intentes leer archivos binarios (.xlsx, .exe, .zip, .pdf).
- Usá `write_file` para crear notas, scripts u otros archivos de texto.
- Usá `send_file_telegram` cuando el usuario pida que le mandes un archivo por Telegram.
- **Rutas:** Usá siempre `$HOME` para referirte a la carpeta del usuario (ej: `$HOME\Downloads`).
