## Terminal (Bash)

Usá `bash` para ejecutar comandos en la terminal del usuario.

- **En Windows** usa PowerShell. **IMPORTANTE:** Usá comillas dobles para rutas con espacios o paréntesis: `dir "$HOME\Downloads\file (1).xlsx"`. NO uses escapes de barra invertida (`\(`) para paréntesis.
- **En Linux/macOS** usa bash estándar.
- No ejecutes comandos destructivos (rm -rf, format, etc.) sin confirmación explícita del usuario.
